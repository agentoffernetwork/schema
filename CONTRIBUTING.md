# Contributing

## Scope

This repository owns machine-readable contracts, types, and validation helpers for AgentOffer Protocol.

## Send Direct PRs For

- Validator improvements
- Schema comments and examples
- Non-breaking tooling changes

## Open An RFC First For

- New schema fields
- Renamed or removed fields
- Semantic changes that affect compatibility

## Validation Expectations

- Keep JSON Schema and TypeScript types aligned
- Update examples when contracts change
