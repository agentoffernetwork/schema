# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the AgentOffer Protocol specification or any related tooling, please report it responsibly.

**Email:** [security@agentoffernetwork.com](mailto:security@agentoffernetwork.com)

Please include:

- A description of the vulnerability
- Steps to reproduce, if applicable
- The potential impact
- Suggested fix, if any

## Response Timeline

- **Acknowledgment:** within 2 business days
- **Initial assessment:** within 5 business days
- **Resolution target:** within 30 days for critical issues

## Scope

This policy covers:

- Protocol specification documents in this repository
- JSON Schema definitions in [`agentoffernetwork/schema`](https://github.com/agentoffernetwork/schema)
- Example payloads in [`agentoffernetwork/examples`](https://github.com/agentoffernetwork/examples)
- Validation tooling published under the `@agentoffernetwork` scope

## Disclosure

We follow coordinated disclosure. Please do not open public issues for security vulnerabilities. We will credit reporters in the advisory unless anonymity is requested.
